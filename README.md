# ImageLoadGUI

Simple interface for opening, viewing and saving cropped subsections of Whole Slide digital pathology images using the (openslide)[openslide.org] python API.

![Screenshot][ss]


Some dependencies:
*Python 2.7
*numpy
*python-imaging-tk
*PIL
*Tkinter

Some planned improvements:
*Batch mode
*Interface refinement
*Mouse-wheel zooming
